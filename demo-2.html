We have tested integrations with React, Vue, Svelte, Angular, and vanilla HTML. DesignKit uses Motion One (~4KB) for spring physics and imperative animations. All animations respect the prefers-reduced-motion media query. Static animations like shimmer and spinner use pure CSS. Yes! All visual properties are controlled via CSS custom properties. Override any --dk-* token in your stylesheet to match your brand. Light and dark themes are included and easy to extend. Every component follows WAI-ARIA patterns. Focus management, keyboard navigation, screen reader support, and reduced motion are built in from the start. The full primitive library is under 30KB gzipped. Section components are tree-shakeable via sub-path imports, so you only pay for what you use.
